
# Overview

This is a sample iOS application to represent three levels of item hierarchy:

* sections
    * first level items
    * CoverFlow/Carousel-like display
* categories
    * second level items
    * central (selected) item is enlarged
* products
    * third level items
    * accessed by swiping the sections and categories away
